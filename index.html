Curso de JS

Javascript é uma linguagem de script criada para aumentar a interatividade entre site e usuário. Os códigos em JS são executados dentro do navegador e dão acesso aos elementos HTML por meio do DOM, que é nada mais que uma representação em forma de árvore da estrutura de um documento HTML ou XML. O DOM organiza a estrutura do documento em uma hierarquia de nós, onde cada nó representa um elemento, atributo, texto ou comentário dentro do documento.

Acesse o curso de JavaScript
